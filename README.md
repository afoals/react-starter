## Getting Started

```sh
npm install
```

Start the local dev server:

```sh
npm start
```

Navigate to **http://localhost:8080/** to view the app.

<br>

## Clean

```sh
npm run clean
```

<br>

## Build for Production

```sh
npm run build
```

<br>

## Publish to Surge.sh
Will require Surge to be installed globally.

```sh
npm run publish
```
