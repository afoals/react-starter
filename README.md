## Getting Started

```sh
$ npm install
```

Start the local dev server:

```sh
$ npm start
```

Navigate to **http://localhost:8080/** to view the app.

<br>

## Build for Production

```sh
$ npm run build
```

**Input:** `src/app.js`

**Output:** `build/app.js`
